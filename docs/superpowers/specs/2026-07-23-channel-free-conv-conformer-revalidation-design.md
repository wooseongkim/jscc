# Channel-Free Conv-Conformer Revalidation Design

## Goal

Re-establish waveform-level feasibility of the frozen SpeechTokenizer continuous
representation and Conv-Conformer JSCC autoencoder before any channel or jammer
work resumes.

## Scope

The implementation provides a dedicated channel-free path with no OFDM, pilots,
allocation, fading, noise, jammer, CSI, equalization, gates, or latent refiner.
SpeechTokenizer remains frozen, while its waveform decoder remains differentiable
with respect to reconstructed continuous representations.

## Model variants

- CF-1 uses the current 50-to-30 and 30-to-50 linear interpolation path with
  1,920 valid complex symbols.
- CF-2 preserves 50 temporal positions with 1,920 valid complex symbols. Each
  layer uses a fixed `[50,5]` complex layout. Ten uniformly spaced frames contain
  four valid symbols and forty frames contain five, yielding exactly 240 symbols
  per layer. Invalid slots are zero and excluded from projection, normalization,
  and counting.
- CF-3 preserves 50 positions and uses eight complex symbols per frame, totaling
  3,200 symbols.
- CF-4 is CF-3 with `d_model=384` and six encoder and decoder blocks.
- CF-5 extends the best eligible CF-1–CF-4 configuration to at least 32,000
  optimizer steps.

The deterministic CF-2 four-symbol frame indices are generated by selecting ten
evenly distributed bins over positions 0–49. The exact indices and full boolean
mask are stored in resolved configuration and checkpoint metadata.

## Training

The target is generated online from real waveforms with the frozen
SpeechTokenizer. No latent cache is allowed. The model sees a neutral observable
state and performs a direct complex-symbol handoff.

The three-stage curriculum is:

1. 4,000 steps of per-layer NMSE plus summed-latent NMSE.
2. 8,000 additional steps with a small multi-resolution STFT term.
3. 8,000 additional steps with a small negative SI-SDR term.

Each active loss term records its scalar value, weighted value, and isolated
gradient norm. Total gradients are clipped. Experiment A may initialize CF-1
when metadata and shapes match; every configuration also supports an explicit
fresh-initialization control.

## Evaluation and selection

Validation uses one fixed deterministic set of 64 unseen-speaker waveforms.
Three independently named checkpoints are selected:

- `best_per_layer_nmse.pt`
- `best_summed_latent_nmse.pt`
- `best_waveform_si_sdr.pt`

Metrics include per-layer and aggregate NMSE, exact decoder-input summed-latent
NMSE/SNR/correlation/power ratio, framewise summed NMSE, SI-SDR, waveform SNR,
and multi-resolution STFT distance.

Baselines are official SpeechTokenizer code reconstruction, continuous feature
sum reconstruction, and per-checkpoint Gaussian perturbations matched separately
to aggregate-layer and summed-latent NMSE.

## Feasibility

A configuration passes only if the fixed 64-utterance evaluation has delta
SI-SDR at least -1 dB, delta waveform SNR at least -1 dB, and STFT ratio no more
than 1.20. Otherwise the result remains
`CHANNEL_FREE_CONV_CONFORMER_NOT_YET_FEASIBLE`.

## Safety

Long experiments are external-only. Existing channel/jammer artifacts remain
untouched. Output directories refuse replacement unless `--overwrite` is
explicitly supplied, and resume validates model, mask, curriculum, dataset, and
checkpoint metadata.
